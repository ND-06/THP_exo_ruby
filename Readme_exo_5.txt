#{} est un "string interpolation operator" 
Cela nous permet d'intégrer dans une chaine de caractéres ( donc un élement de type string avec guillemets ouvrantes et fermantes ) des opérations mathématiques avec des nombres. Les nombres et opérateurs ( de calcul ou de comparaison ) sont à mettre dans les accolades et aprés le # . Cela nous permet d'avoir en retour à la fois la phrase que l'on a mis dans notre
code mais également le résultat de ce que l'on a mis dans les accolades. Sans ces symboles #{} , on aurait eu
juste une phrase et pas de calcul.
Pour résumer, cela nous permet donc de lancer un calcul au sein même d'une phrase ;d'une chaine de caractéres.
