gets.chomp permet directement d'intéragir avec l'utilisateur en lui demandant, dans le cas de notre
code, son blase ( son nom quoi ). get.chomp a la même fonction que le "Prompt" de Javascript:
c'est a dire ouvrir une boite de dialogue demandant à l'utilisateur de retourner une donnée 
( un nombre , un mot, ou une phrase ). Une fois la donnée retournée par l'utilisateur, cette
dernière est directement stockée dans la variable pour laquelle a été assignée le get.chomp 
( ici user_name )
